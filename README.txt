# HOW TO RUN #
To run the matlab project, (assuming you have matlab installed) set your IDE directory/current folder to the folder containing all of the matlab scripts. Right click the source folder and select "Add directory and sub directories".

# TO START #
Either open the MainRayTracer.m file and press the run button at the top of the IDE. OR in the console type "mainRayTracer" without the quotes
